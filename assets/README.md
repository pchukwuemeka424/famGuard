# Assets Directory

This directory contains all static assets for the SafeZone app.

## Directory Structure

- `images/` - Image assets used throughout the app
- `icon.png` - App icon (optional, can be added later)
- `splash.png` - Splash screen image (optional, can be added later)

## Notes

- The `images/` folder is required for Metro bundler
- App icon and splash screen are optional and can be added when needed
- All assets in this directory will be bundled with the app
